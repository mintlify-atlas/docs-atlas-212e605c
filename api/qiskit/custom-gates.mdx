Custom quantum gates provided by Qiskit Superstaq for hardware-specific operations and optimizations.

## AceCR

Active Cancellation Echoed Cross Resonance gate, supporting polarity switches and sandwiches.

```python
class AceCR(qiskit.circuit.Gate)
```

The typical AceCR in the literature is a positive half-CR, then X on "Z side", then negative half-CR (where "Z side" and "X side" refer to the two sides of the underlying ZX interactions).

### Constructor

```python
AceCR(
    rads: str | float = np.pi / 2,
    sandwich_rx_rads: float = 0,
    label: str | None = None
)
```

<ParamField path="rads" type="str | float">
Angle of rotation for CR gate (i.e., twice the angle for each echoed half-CR). Can be a float value or "+-" (equivalent to ѕА/2) or "-+" (equivalent to -ѕА/2). Defaults to ѕА/2.
</ParamField>

<ParamField path="sandwich_rx_rads" type="float">
Angle of rotation for an rx gate applied to the "X side" simultaneously with the X gate on the "Z side". Defaults to 0.
</ParamField>

<ParamField path="label" type="str | None">
An optional label for the constructed gate.
</ParamField>

### Methods

#### inverse

```python
inverse() -> AceCR
```

Inverts the AceCR gate.

**Returns:** The inverse AceCR gate.

## ZZSwapGate

The ZZ-SWAP gate, which performs the ZZ-interaction followed by a SWAP.

```python
class ZZSwapGate(qiskit.circuit.Gate)
```

ZZ-SWAPs are useful for applications like QAOA or Hamiltonian Simulation, particularly on linear- or low-connectivity devices. See https://arxiv.org/pdf/2004.14970.pdf for an application of ZZ-SWAP networks.

The unitary for a ZZ-SWAP gate parametrized by ZZ-interaction angle ќЄ is:

```
вФМ         вФР
вФВ 1  .  .  . вФВ
вФВ .  . e^iќЄ . вФВ
вФВ . e^iќЄ . . вФВ
вФВ .  .  . 1 вФВ
вФФ         вФШ
```

For ќЄ = 0, the ZZ-SWAP gate is just an ordinary SWAP.

### Constructor

```python

    theta: float,
    label: str | None = None
)
```

<ParamField path="theta" type="float">
The ZZ-interaction angle in radians.
</ParamField>

<ParamField path="label" type="str | None">
An optional label for the constructed gate.
</ParamField>

### Methods

#### inverse

```python
inverse() -> ZZSwapGate
```

Inverts the ZZ-SWAP gate.

**Returns:** The inverse ZZ-SWAP gate.

## StrippedCZGate

The Stripped CZ gate is a regular CZ gate when the rz angle = 0.

```python
class StrippedCZGate(qiskit.circuit.Gate)
```

It is the gate that is actually being performed by Sqale, and it is corrected into a CZ gate by RZ gates afterwards if the rz angle is nonzero.
