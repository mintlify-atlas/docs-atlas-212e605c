## What is Superstaq?

Superstaq is a quantum software platform that enables you to write quantum programs in Cirq or Qiskit and target a variety of quantum computers and simulators. The platform is optimized across the quantum stack and provides:

- **Write-once-target-all**: Write your quantum circuit once and run it on multiple quantum backends
- **Pulse level optimizations**: Advanced optimizations at the pulse level for better performance
- **Multi-framework support**: Native support for both Qiskit and Cirq quantum programming frameworks
- **Unified API**: Access to various quantum computers and simulators through a single interface

<Note>
  Superstaq supports Python versions 3.9, 3.10, 3.11, 3.12, 3.13, and 3.14.
</Note>

## Available packages

Superstaq offers three Python packages to fit your workflow:

### qiskit-superstaq

For developers using the Qiskit framework. Access Superstaq's features while working with familiar Qiskit circuits and backends.

### cirq-superstaq

For developers using the Cirq framework. Integrate Superstaq capabilities into your Cirq quantum programs.

### general-superstaq

Core functionality shared across frameworks. This package is used internally by both qiskit-superstaq and cirq-superstaq.
