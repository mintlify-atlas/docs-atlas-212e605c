This guide will help you submit your first quantum circuit to Superstaq. We'll create a simple Bell state circuit and run it on a simulated quantum backend.

<Note>
  Before you begin, make sure you have:
  - Python 3.9 or higher installed
  - A Superstaq API token from [superstaq.infleqtion.com](https://superstaq.infleqtion.com)
</Note>

## Choose your framework

Superstaq supports both Qiskit and Cirq. Select the framework you prefer:

<Tabs>
  <Tab title="Qiskit">
    
<Steps>
  <Step title="Install qiskit-superstaq">
    Install the package using pip:
    
    ```bash
    pip install qiskit-superstaq
    ```
  </Step>

  <Step title="Create a Bell state circuit">
    Create a simple quantum circuit that produces a Bell state:
    
    ```python
    import qiskit
    import qiskit_superstaq as qss

    # Create a quantum circuit with 2 qubits and 2 classical bits
    qc = qiskit.QuantumCircuit(2, 2)
    qc.h(0)  # Apply Hadamard gate to qubit 0
    qc.cx(0, 1)  # Apply CNOT gate with control=0, target=1
    qc.measure(0, 0)  # Measure qubit 0 into classical bit 0
    qc.measure(1, 1)  # Measure qubit 1 into classical bit 1

    print(qc.draw())
    ```
  </Step>

  <Step title="Set up the Superstaq provider">
    Initialize the Superstaq provider with your API token:
    
    ```python
    # Replace with your token from https://superstaq.infleqtion.com
    token = "your_superstaq_token_here"
    
    superstaq = qss.SuperstaqProvider(token)
    ```
    
    <Warning>
      Keep your API token secure and never commit it to version control. Consider using environment variables.
    </Warning>
  </Step>

  <Step title="Submit and run your circuit">
    Submit your circuit to a backend and retrieve the results:
    
    ```python
    # Get the IBM Fez QPU backend
    backend = superstaq.get_backend("ibmq_fez_qpu")
    
    # Submit the circuit with dry-run method (available for free trial users)
    job = backend.run(qc, shots=100, method="dry-run")
    
    # Get and print the results
    print(job.result().get_counts())
    ```
    
    <Tip>
      The `method="dry-run"` parameter simulates the circuit execution and is available to all users, including those on the free trial.
    </Tip>
  </Step>
</Steps>

  </Tab>
  <Tab title="Cirq">
    
<Steps>
  <Step title="Install cirq-superstaq">
    Install the package using pip:
    
    ```bash
    pip install cirq-superstaq
    ```
  </Step>

  <Step title="Create a Bell state circuit">
    Create a simple quantum circuit that produces a Bell state:
    
    ```python
    import cirq
    import cirq_superstaq as css

    # Define qubits
    q0 = cirq.LineQubit(0)
    q1 = cirq.LineQubit(1)

    # Create the circuit
    circuit = cirq.Circuit(
        cirq.H(q0),  # Apply Hadamard gate to q0
        cirq.CNOT(q0, q1),  # Apply CNOT gate
        cirq.measure(q0)  # Measure q0
    )
    ```
  </Step>

  <Step title="Set up the Superstaq service">
    Initialize the Superstaq service with your API token:
    
    ```python
    # Replace with your token from https://superstaq.infleqtion.com
    service = css.Service(
        api_key="your_superstaq_token_here",
        verbose=True
    )
    ```
    
    <Warning>
      Keep your API token secure and never commit it to version control. Consider using environment variables.
    </Warning>
  </Step>

  <Step title="Submit and run your circuit">
    Submit your circuit to a backend and retrieve the results:
    
    ```python
    # Submit the circuit to IBM Fez QPU with dry-run method
    job = service.create_job(
        circuit,
        repetitions=100,
        target="ibmq_fez_qpu",
        method="dry-run"
    )
    
    # Check job status and get results
    print("Job status:", job.status())
    print(job.counts(0))
    ```
    
    <Tip>
      The `method="dry-run"` parameter simulates the circuit execution and is available to all users, including those on the free trial.
    </Tip>
  </Step>
</Steps>

  </Tab>
</Tabs>

## Understanding the output

For a Bell state circuit, you should see results showing the two entangled qubits measured in correlated states. The output will be a dictionary with counts for each measurement outcome:

```python
{'00': 52, '11': 48}  # Example output showing entanglement
```

The Bell state creates quantum entanglement between the two qubits, so they should always be measured in the same state (both 0 or both 1).
